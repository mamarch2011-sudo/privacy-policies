## 📝 Kako Dodati Novu Aplikaciju

1. Kreiraj novi HTML fajl (npr. `nova-app-android.html`)
2. Koristi postojeće fajlove kao template
3. Dodaj link u `index.html`
4. Commit i push na GitHub
5. GitHub Pages će automatski objaviti novu stranicu
